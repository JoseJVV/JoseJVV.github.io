# Portfolio de JoseJVV – Minimal Mistakes

Estructura inicial del portfolio basada en el tema **Minimal Mistakes** para GitHub Pages,
con una estética de **fantasía oscura** y una paleta de colores:

- Azul principal (tipo ArtStation): `#0097f5`
- Azul hielo: `#7fd4ff`
- Negro azulado profundo: `#05060a`
- Blanco suave: `#f5f5f5`

## Secciones principales

- **Inicio**: Sobre mí, resumen CV y proyectos profesionales destacados.
- **Ilustración**: Proyectos 2D, organizados en Cómic, Diseño gráfico, Concept Art y Diseño de personajes.
- **3D**: Proyectos de modelado 3D, organizados en Personajes, Environment, Videojuegos e Impresión 3D.
- **Contacto**: Datos de contacto y enlaces a redes.

## Carpeta PROYECTOS

Utiliza la carpeta `PROYECTOS` para subir tus imágenes y archivos de cada proyecto:

- `PROYECTOS/ilustracion`
- `PROYECTOS/3d`

Dentro puedes crear una carpeta por proyecto y organizar tus recursos como prefieras.
