# Diseño de personajes

Personajes con personalidad, silueta clara y lenguaje visual definido.

Aquí podrás subir tus proyectos de **Diseño de personajes**.  
Te recomiendo crear una carpeta dentro de `PROYECTOS/ilustracion` para cada proyecto, por ejemplo:

- `PROYECTOS/ilustracion/diseño-de-personajes-proyecto-1`
- `PROYECTOS/ilustracion/diseño-de-personajes-proyecto-2`

Dentro de cada carpeta puedes guardar:
- Imágenes finales (PNG/JPG)
- Bocetos
- Turnarounds
- Notas en un archivo `.txt` o `.md`
