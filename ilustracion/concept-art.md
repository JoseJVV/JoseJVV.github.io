# Concept Art

Exploración visual para mundos, escenas y momentos clave.

Aquí podrás subir tus proyectos de **Concept Art**.  
Te recomiendo crear una carpeta dentro de `PROYECTOS/ilustracion` para cada proyecto, por ejemplo:

- `PROYECTOS/ilustracion/concept-art-proyecto-1`
- `PROYECTOS/ilustracion/concept-art-proyecto-2`

Dentro de cada carpeta puedes guardar:
- Imágenes finales (PNG/JPG)
- Bocetos
- Turnarounds
- Notas en un archivo `.txt` o `.md`
