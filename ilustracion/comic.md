# Cómic

Páginas, portadas y viñetas con narrativa visual y ritmo.

Aquí podrás subir tus proyectos de **Cómic**.  
Te recomiendo crear una carpeta dentro de `PROYECTOS/ilustracion` para cada proyecto, por ejemplo:

- `PROYECTOS/ilustracion/cómic-proyecto-1`
- `PROYECTOS/ilustracion/cómic-proyecto-2`

Dentro de cada carpeta puedes guardar:
- Imágenes finales (PNG/JPG)
- Bocetos
- Turnarounds
- Notas en un archivo `.txt` o `.md`
