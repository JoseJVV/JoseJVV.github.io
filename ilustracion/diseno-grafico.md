# Diseño gráfico

Carteles, portadas, branding y piezas promocionales.

Aquí podrás subir tus proyectos de **Diseño gráfico**.  
Te recomiendo crear una carpeta dentro de `PROYECTOS/ilustracion` para cada proyecto, por ejemplo:

- `PROYECTOS/ilustracion/diseño-gráfico-proyecto-1`
- `PROYECTOS/ilustracion/diseño-gráfico-proyecto-2`

Dentro de cada carpeta puedes guardar:
- Imágenes finales (PNG/JPG)
- Bocetos
- Turnarounds
- Notas en un archivo `.txt` o `.md`
